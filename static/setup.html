推理服务配置
珊瑚菌使用本地语言模型处理日常对话。请填写推理服务的地址和模型名称。
LM Studio 默认地址：http://localhost:1234/v1/chat/completions
推理服务地址 *
兼容 OpenAI API 格式的推理服务地址
模型名称 *
必须与推理服务中实际加载的模型名称完全一致（区分大小写）
嵌入模型配置
嵌入模型将文本转换为语义向量，是记忆检索功能的基础。请填写已下载的模型文件夹的完整绝对路径。
推荐模型：Qwen/Qwen3-Embedding-0.6B（约 600MB）
嵌入模型文件夹路径 *
填写包含 config.json 的模型文件夹路径，不是具体文件路径。
路径分隔符请统一使用正斜杠 / （兼容所有操作系统）。
下载方法：访问 hf-mirror.com 下载全部文件后放入同一文件夹。
人格配置
珊瑚菌的性格、说话风格、身份设定都在这里定义。已为你准备了默认配置，可直接跳过此步骤使用。
提示： 此步骤完全可选。默认配置已包含完整的人格设定，如果你暂时不想修改，点「下一步」直接跳过即可。修改后点「保存人格」生效。
persona.toml 内容
修改 [identity] 中的 assistant_name/user_name，以及 [blocks] 中的 A_persona/E_rules 来自定义人格。
可选配置
以下配置项均有合理默认值，可以直接跳过。需要使用云端 API 或天气功能时再填写。
云端 API Key （可选）
用于调用云端模型处理复杂推理任务。留空则只使用本地模型。
天气查询城市 （可选）
英文城市名，用于天气感知功能。留空时通过 IP 自动定位。
服务端口 （可选）
珊瑚菌服务监听的端口号，默认 8000。端口被占用时需修改。
